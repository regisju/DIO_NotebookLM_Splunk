# 📘 Miniguia de Estudos

## 📌 O que é Splunk?

O Splunk é uma plataforma utilizada para coleta, monitoramento, indexação e análise de dados gerados por sistemas, aplicações e dispositivos.

A ferramenta é amplamente utilizada em ambientes de segurança da informação para análise de logs e monitoramento de eventos.

---

## 📌 O que é SIEM?

SIEM (Security Information and Event Management) é uma solução responsável por centralizar logs e eventos de diferentes dispositivos e sistemas, permitindo monitoramento, detecção de ameaças e investigação de incidentes.

---

## 📌 Correlação de Eventos

A correlação de eventos consiste em relacionar diferentes logs e atividades para identificar padrões suspeitos ou comportamentos maliciosos.

Exemplo:

* Múltiplas tentativas de login;
* Acesso fora do horário padrão;
* Alterações incomuns em sistemas;
* Eventos simultâneos em diferentes servidores.

---

## 📌 Principais Funcionalidades do Splunk

* Coleta de logs;
* Indexação de dados;
* Pesquisa e análise;
* Dashboards interativos;
* Monitoramento em tempo real;
* Correlação de eventos;
* Geração de alertas.

---

## 📌 Benefícios do Splunk

* Centralização de logs;
* Melhor visibilidade do ambiente;
* Detecção rápida de ameaças;
* Automação de monitoramento;
* Apoio em investigações de segurança.

# 📚 Glossário

| Termo                 | Definição                                                |
| --------------------- | -------------------------------------------------------- |
| SIEM                  | Security Information and Event Management                |
| Log                   | Registro de eventos gerados por sistemas                 |
| Splunk                | Plataforma de monitoramento e análise de dados           |
| Correlação de Eventos | Relação entre múltiplos eventos para identificar padrões |
| Dashboard             | Painel visual de monitoramento                           |
| Indexação             | Processo de organização de dados para pesquisa           |
